

『sodate』




　●概要
　
　ネットワーク更新を一動作で終わらせる為のツール。
　具体的には、
　
　・updates2.dauの作成
　・ftpでアップロード
　・narアーカイブの作成（＆毎日のバックアップ）
　
　をします。
　
　profileフォルダや各栞のセーブデータなど、
　更新の不要なファイル・フォルダは対象とせず（自動除外して）、
　必要なファイルだけをアップロードします。
　
　
　

　●設置

　　+ 何か
　　　+ ghost
　　　　+ あなたのゴースト名
　　　　　+ ghost
　　　　　+ shell
　　　　　- sodate.exe
　　　　　- sodate_setup.exe

　
　　+ はフォルダ。- は設置するファイル。
　　

　●設定
　
　　sodate_setup.exe を実行してください。
　　
　　わかんないところでは　？　ボタンを押してください。
　　説明不足な点があれば指摘していただけると幸いです。
　　
　　各種設定を行って、OKボタンを押すと、
　　sodate.datというファイルが生成されます。
　　　
　
　●実行
　
　　sodate.exeを実行してください。

　　sodate_log.txtというログファイルを作ります。
　　表示したものと同内容です。
